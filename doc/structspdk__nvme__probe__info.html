Detailed Description

NVMe controller information provided during spdk_nvme_probe().

struct spdk_pci_id spdk_nvme_probe_info::pci_id

§ subnqn

char spdk_nvme_probe_info::subnqn[SPDK_NVMF_NQN_MAX_LEN+1]

Subsystem NQN.

If this is not an NVMe over Fabrics controller, this field will be a zero-length string.

§ traddr

char spdk_nvme_probe_info::traddr[SPDK_NVMF_TRADDR_MAX_LEN+1]

Transport address of the NVMe over Fabrics target.

For transports which use IP addressing (e.g. RDMA), this will be an IP-based address.

§ trsvcid

char spdk_nvme_probe_info::trsvcid[SPDK_NVMF_TRSVCID_MAX_LEN+1]

Transport service identifier.

For transports which use IP addressing (e.g. RDMA), this field will be the port number.

§ trtype

enum spdk_nvmf_trtype spdk_nvme_probe_info::trtype

NVMe over Fabrics transport type.

This field will be 0 if this is not an NVMe over Fabrics controller.
