Detailed Description

NVMe over Fabrics discovery parameters.

This structure must be provided when connecting to remote NVMe controllers via NVMe over Fabrics.

Field Documentation

§ traddr

const char* spdk_nvme_discover_info::traddr

Transport address of the NVMe over Fabrics discovery service.

For transports which use IP addressing (e.g. RDMA), this should be an IP-based address.
